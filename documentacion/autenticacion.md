# Instructivo autenticacion y acceso de usuarios

El sistema tiene acceso restringido solo para usuarios autenticados. Un usuario no autenticado solo tiene acceso a la página de inicio, y al módulo de autenticación, que incluye login, registro, y la funcionalidad de recuperar contraseña.

Desde el Home, es posible acceder al login y al registro de un usuario nuevo

## Registro

<img width="387" height="307" alt="imagen" src="https://github.com/user-attachments/assets/ec9c88ce-4521-4960-8181-c8e36e65c1d0" />

El usuario puede registrarse en el sistema como Estudiante o como Empresa, completando la información requerida en cada caso.
Se debe tener en cuenta que el correo electrónico y el DNI que se usen para crear el usuario no deben estar ya en uso, o el sistema mostrará un mensaje de error y no permitirá continuar con el registro

## Login

<img width="402" height="405" alt="imagen" src="https://github.com/user-attachments/assets/1be716ff-aaa5-450a-b1a7-2973294abc57" />

## Recuperar contraseña
