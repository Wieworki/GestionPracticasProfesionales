## Ofertas disponibles

<img width="1326" height="594" alt="imagen" src="https://github.com/user-attachments/assets/e766516d-2315-4265-9332-54dc05caa1ad" />

Desde la opción "Ver ofertas disponibles" el usuario puede acceder a un listado con todas las ofertas a las que es posible postularse, pudiendo acceder al detalle de las mismas

### Postularse a oferta

<img width="633" height="311" alt="imagen" src="https://github.com/user-attachments/assets/9c224cf8-36ff-4ac5-8a74-eb1d160d7888" />

Dentro del detalle de una oferta de práctica profesional, si el usuario aún no se postuló a la misma, tendrá disponible el botón "Postularse".
**Si el usuario ya se postuló a la misma anteriormente, este botón no será visible.**

### Anular postulación

<img width="633" height="375" alt="imagen" src="https://github.com/user-attachments/assets/b15920ea-73fe-405b-9127-63b2174638f4" />

Dentro del detalle de una oferta de práctica profesional, si el usuario realizó una postulación a la misma, tendrá disponible en botón "Anular postulación".
**Si el usuario no tiene una postulación hecha, este botón no será visible.**

### Confirmar postulación

<img width="639" height="515" alt="imagen" src="https://github.com/user-attachments/assets/025157f3-a352-4283-813a-60c301195298" />

En el caso de que la empresa seleccione al usuario como el elegido para realizar la práctica profesional, el usuario tendrá la opción de "Confirmar postulación", para confirmarle a la empresa de que está disponible para realizarla.
Este botón **solo está disponible si la empresa seleccionó su postulación**.
